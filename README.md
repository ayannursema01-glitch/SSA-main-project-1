# SSA-main-project-1
SSA (Smart Student Assistant) is an AI study co-pilot. • PDF Tools: Drag-and-drop PDFs to instantly extract summaries, flashcards, and mock quizzes. • Gamification: Stay motivated by earning XP, leveling up, and maintaining streaks. • Focus Room: Block out distractions with an immersive study timer. Streamline learning and ace your exams.
